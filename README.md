# Template PlatformIO — Arduino Nano

Template pour démarrer rapidement un projet PlatformIO basé sur Arduino Nano.

## Structure

- `src/` : Code source principal
- `include/` : Headers
- `lib/` : Librairies locales
- `docs/` : Documentation (schémas, câblage)
- `platformio.ini` : Configuration PlatformIO
